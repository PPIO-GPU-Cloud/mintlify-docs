## 派欧算力云

派欧算力云是 PPIO 旗下最新产品。

派欧算力云秉承提供高性价比算力服务的宗旨，推出了 GPU 弹性算力产品，通过提供多样化的 GPU 、优化的性能以及充足的算力资源，为新一代生成式人工智能、云端渲染、机器学习和加速计算等领域提供高性价比的算力支持。

## GPU 容器云介绍

GPU 容器云在普通容器云上附加了 GPU 加速卡，提供超强计算能力的同时，保持灵活发放和按需使用的特性。适合 AI 训练、AI 推理、科学计算、视频渲染等场景。

GPU 容器云支持与普通容器云相同的使用和管理方式，如结合私有网络、安全组、密钥对、负载均衡等服务，实现丰富业务架构，方便管理。

除了普通容器云，派欧算力云还提供高性能计算 GPU 裸金属，提供高性能计算、超大内存、高吞吐量的计算能力，显著提升网络性能，适用于高性能计算、人工智能、机器学习等业务场景。
